Texniki sahədə, xüsusən də şəbəkələşmədə, çoxlu sayda termin var. Əsas terminologiyanın Azərbaycanca tərcüməsi və izahı ilə yaradılmış siyahı aşağıdakı kimidir. Bu, gələcək modullarda müzakirə ediləcək mövzuları başa düşmək üçün zəmin yaradır.

---

## Şəbəkənin Əsas Terminologiyası 🔑

Aşağıdakı cədvəldə informasiya texnologiyaları və şəbəkə sahəsində ən çox rast gəlinən protokolların adları, abreviaturları və qısa təsvirləri verilib:

| Protokol | Abreviatura | Təsvir |
| :--- | :--- | :--- |
| **Wired Equivalent Privacy** | **WEP** | Simsiz şəbəkələri qorumaq üçün istifadə edilən bir təhlükəsizlik protokol növüdür (artıq köhnəlmişdir). |
| **Secure Shell** | **SSH** | Uzaq sistemə daxil olmaq və əmrləri yerinə yetirmək üçün istifadə edilən təhlükəsiz şəbəkə protokoludur. |
| **File Transfer Protocol** | **FTP** | Faylları bir sistemdən digərinə ötürmək üçün istifadə edilən şəbəkə protokoludur. |
| **Simple Mail Transfer Protocol** | **SMTP** | E-poçtları göndərmək və qəbul etmək üçün istifadə edilən protokoldur. |
| **Hypertext Transfer Protocol** | **HTTP** | İnternet üzərindən məlumat göndərmək və qəbul etmək üçün istifadə edilən *klient-server* protokoludur. |
| **Server Message Block** | **SMB** | Şəbəkədə faylları, printerləri və digər resursları paylaşmaq üçün istifadə edilən protokoldur. |
| **Network File System** | **NFS** | Şəbəkə üzərindən fayllara daxil olmaq üçün istifadə edilən protokoldur. |
| **Simple Network Management Protocol** | **SNMP** | Şəbəkə cihazlarını idarə etmək üçün istifadə edilən protokoldur. |
| **Wi-Fi Protected Access** | **WPA** | Simsiz şəbəkələri icazəsiz girişdən qorumaq üçün parol istifadə edən simsiz təhlükəsizlik protokoludur. |
| **Temporal Key Integrity Protocol** | **TKIP** | Simsiz şəbəkələrdə istifadə olunan, lakin daha az təhlükəsiz olan bir təhlükəsizlik protokoludur. |
| **Network Time Protocol** | **NTP** | Şəbəkədəki kompüterlərin vaxtını sinxronlaşdırmaq üçün istifadə edilir. |
| **Virtual Local Area Network** | **VLAN** | Şəbəkəni çoxsaylı məntiqi şəbəkələrə seqmentləşdirmə üsuludur. |
| **VLAN Trunking Protocol** | **VTP** | Birdən çox kommutatoru (*switch*) əhatə edən virtual LAN-ı (VLAN) qurmaq və saxlamaq üçün istifadə edilən Qat 2 protokoludur. |
| **Routing Information Protocol** | **RIP** | Yerli və geniş ərazi şəbəkələrində (**LAN/WAN**) istifadə olunan *məsafə-vektor* marşrutlaşdırma protokoludur. |
| **Open Shortest Path First** | **OSPF** | Bir İnternet Protokol (IP) şəbəkəsində tək bir *Avtonom Sistem* (*AS*) daxilində trafikin marşrutlaşdırılması üçün *daxili şlüz* protokoludur (**IGP**). |
| **Interior Gateway Routing Protocol** | **IGRP** | Avtonom sistemlər daxilində marşrutlaşdırma üçün nəzərdə tutulmuş **Cisco**-ya məxsus *daxili şlüz* protokoludur. |
| **Enhanced Interior Gateway Routing Protocol** | **EIGRP** | Şəbəkə daxilində IP trafikini marşrutlaşdırmaq üçün istifadə edilən *təkmilləşdirilmiş məsafə-vektor* marşrutlaşdırma protokoludur. |
| **Pretty Good Privacy** | **PGP** | E-poçtları, faylları və digər məlumat növlərini qorumaq üçün istifadə edilən şifrələmə proqramıdır. |
| **Network News Transfer Protocol** | **NNTP** | İnternetdə xəbər qruplarında mesajların yayılması və əldə edilməsi üçün istifadə edilən protokoldur. |
| **Cisco Discovery Protocol** | **CDP** | Şəbəkə administratorlarına şəbəkəyə qoşulmuş Cisco cihazlarını aşkar etməyə və idarə etməyə imkan verən Cisco Systems tərəfindən hazırlanmış xüsusi (*proprietary*) protokoldur. |
| **Hot Standby Router Protocol** | **HSRP** | Router və ya digər şəbəkə cihazının sıradan çıxması halında ehtiyat təmin etmək üçün Cisco routerlərində istifadə edilən protokoldur. |
| **Virtual Router Redundancy Protocol** | **VRRP** | İştirak edən hostlara mövcud *İnternet Protokol (IP) routerlərinin avtomatik təyin edilməsini* təmin etmək üçün istifadə edilən protokoldur. |
| **Spanning Tree Protocol** | **STP** | Qat 2 Ethernet şəbəkələrində **dövrəsiz** (*loop-free*) topologiyanı təmin etmək üçün istifadə edilən şəbəkə protokoludur. |
| **Terminal Access Controller Access-Control System** | **TACACS** | Şəbəkə girişinə görə *mərkəzləşdirilmiş autentifikasiya, avtorizasiya və uçot* (**AAA**) təmin edən protokoldur. |
| **Session Initiation Protocol** | **SIP** | IP şəbəkəsi üzərindən real vaxt rejimində səs, video və multimedia sessiyalarını qurmaq və ləğv etmək üçün istifadə edilən *siqnal protokolu*dur. |
| **Voice Over IP** | **VOIP** | İnternet üzərindən telefon zənglərinin edilməsinə imkan verən bir texnologiyadır. |
| **Extensible Authentication Protocol** | **EAP** | Parollar, rəqəmsal sertifikatlar və birdəfəlik parollar kimi *çoxsaylı autentifikasiya metodlarını* dəstəkləyən bir autentifikasiya çərçivəsidir. |
| **Lightweight Extensible Authentication Protocol** | **LEAP** | Cisco Systems tərəfindən hazırlanmış xüsusi simsiz autentifikasiya protokoludur. **Point-to-Point Protocol (PPP)**-də istifadə olunan **EAP**-a əsaslanır. |
| **Protected Extensible Authentication Protocol** | **PEAP** | Simsiz şəbəkələr və digər şəbəkə növləri üçün **şifrələnmiş tunel** təmin edən təhlükəsizlik protokoludur. |
| **Systems Management Server** | **SMS** | Təşkilatlara şəbəkələrini, sistemlərini və mobil cihazlarını idarə etməyə kömək edən *sistem idarəetmə həllidir*. |
| **Microsoft Baseline Security Analyzer** | **MBSA** | **Windows** kompüterlərində, şəbəkələrində və sistemlərində potensial təhlükəsizlik zəifliklərini aşkar etmək üçün istifadə edilən Microsoft-dan pulsuz təhlükəsizlik vasitəsidir. |
| **Supervisory Control and Data Acquisition** | **SCADA** | İstehsalat, enerji istehsalı və su/tullantıların təmizlənməsi kimi sənaye proseslərini monitorinq etmək və idarə etmək üçün istifadə edilən *sənaye nəzarət sistemi* növüdür. |
| **Virtual Private Network** | **VPN** | İstifadəçilərə internet üzərindən başqa bir şəbəkəyə **təhlükəsiz, şifrələnmiş əlaqə** yaratmağa imkan verən bir texnologiyadır. |
| **Internet Protocol Security** | **IPsec** | Şəbəkə üzərindən **təhlükəsiz, şifrələnmiş rabitə** təmin etmək üçün istifadə edilən protokoldur. VPN-lərdə təhlükəsiz tunel yaratmaq üçün istifadə olunur. |
| **Point-to-Point Tunneling Protocol** | **PPTP** | Uzaqdan giriş üçün təhlükəsiz, şifrələnmiş tunel yaratmaq üçün istifadə edilən protokoldur. |
| **Network Address Translation** | **NAT** | Şəxsi şəbəkədəki çoxsaylı cihazlara **tək bir ictimai IP ünvanı** istifadə edərək internetə qoşulmağa imkan verən bir texnologiyadır. |
| **Carriage Return Line Feed** | **CRLF** | Müəyyən mətn faylı formatları üçün **sətirin sonunu** və yenisinin başlanğıcını göstərmək üçün iki nəzarət simvolunu birləşdirir. |
| **Asynchronous JavaScript and XML** | **AJAX** | **JavaScript** və **XML/JSON** istifadə edərək dinamik veb səhifələr yaratmağa imkan verən veb inkişaf texnikasıdır. |
| **Internet Server Application Programming Interface** | **ISAPI** | **API** dəstindən istifadə edərək veb serverlər üçün *məhsuldarlıq yönümlü veb uzantıları* yaratmağa imkan verir. |
| **Uniform Resource Identifier** | **URI** | İnternetdə bir resursu müəyyən etmək üçün istifadə edilən bir sintaksisdir. |
| **Uniform Resource Locator** | **URL** | **URI**-nin bir alt çoxluğudur ki, protokolu və domen adını daxil etməklə internetdə bir veb səhifəni və ya başqa bir resursu müəyyən edir. |
| **Internet Key Exchange** | **IKE** | İki kompüter arasında **təhlükəsiz əlaqə** qurmaq üçün istifadə edilən protokoldur. VPN-lərdə autentifikasiya və şifrələmə təmin etmək üçün istifadə olunur. |
| **Generic Routing Encapsulation** | **GRE** | Bu protokol **VPN tuneli** daxilində ötürülən məlumatları **kapsullamaq** (*encapsulate*) üçün istifadə olunur. |
| **Remote Shell** | **RSH** | **Unix** altında uzaq kompüterdə əmrləri və proqramları yerinə yetirməyə imkan verən bir proqramdır. |

Bu terminologiya siyahısı şəbəkə sahəsinə giriş üçün **əsas təməl** təşkil edir. Yeni protokollar öyrəndikcə bu siyahını genişləndirməyiniz tövsiyə olunur.
